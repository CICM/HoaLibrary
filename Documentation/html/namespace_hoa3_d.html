The associated Legendre polynomials.

The associated Legendre polynomial . are used to compute the spherical harmonic coefficients where l is the band and the m is the argument of a spherical harmonic and x is the cosinus of the elevation. You should prefer to use the spherical harmonic elevation function that uses a normalization. It uses three recurrence formulas :
P(l, l)(x) = (-1)^l * (2l - 1)!! * (1 - x * x )^0.5,
P(l + 1, l)(x) = x * (2l + 1) * P(l, l) and
P(l + 1, m)(x) = ((2l + 1) * x * P(m, l) - (l + m) * P(m, l - 1)) / (l - m + 1).

Parameters
bandThe band of the spherical harmonic.
argumentThe argument of the spherical harmonic.
xThe cosinus of the elevation.
Returns
The function return the associated Legendre polynomial P(l, m)(x).
