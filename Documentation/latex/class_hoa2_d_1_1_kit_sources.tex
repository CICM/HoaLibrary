\subsection{Detailed Description}
The kit to spatialize lught points sources. 

The \hyperlink{class_hoa2_d_1_1_kit_sources}{Kit\-Sources} is an all-\/in-\/one class that owns a \hyperlink{class_hoa2_d_1_1_map}{Map}, an \hyperlink{class_hoa2_d_1_1_optim}{Optim}, a \hyperlink{class_hoa2_d_1_1_decoder_multi}{Decoder\-Multi}, a \hyperlink{class_hoa2_d_1_1_meter}{Meter} and a \hyperlink{class_hoa2_d_1_1_sources_manager}{Sources\-Manager} and that can be used to spatialize several sources. It allows to dynamicaly change the classes parameters.
