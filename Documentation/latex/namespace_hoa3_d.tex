The associated Legendre polynomials. 

The associated Legendre polynomial $\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$. $P(l, m)$ are used to compute the spherical harmonic coefficients where l is the band and the m is the argument of a spherical harmonic and x is the cosinus of the elevation. You should prefer to use the spherical harmonic elevation function that uses a normalization. It uses three recurrence formulas \-: \par
P(l, l)(x) = (-\/1)$^\wedge$l $\ast$ (2l -\/ 1)!! $\ast$ (1 -\/ x $\ast$ x )$^\wedge$0.5, \par
P(l + 1, l)(x) = x $\ast$ (2l + 1) $\ast$ P(l, l) and \par
P(l + 1, m)(x) = ((2l + 1) $\ast$ x $\ast$ P(m, l) -\/ (l + m) $\ast$ P(m, l -\/ 1)) / (l -\/ m + 1).
