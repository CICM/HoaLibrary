version Beta 2.0
------------------------

Max : For an overview of hoa.library and new features in version Beta 2.0, please refer to the 'hoa.overview' patch in the extra folder.


• New Objects and Features:

 - new 3-D objects : hoa.3d.encoder~, hoa.3d.decoder~, hoa.3d.map~, hoa.3d.optim~, hoa.3d.scope~, hoa.3d.vector~, hoa.3d.wider~

 - hoa.process~ : new object to manage patcher based multichannel processing.

 - hoa.in~ : signal inlet for hoa.process~ object.

 - hoa.out~ : signal outlet for hoa.process~ object.

 - hoa.in : control inlet for hoa.process~ object.

 - hoa.out : control outlet for hoa.process~ object.

 - hoa.thisprocess~ : report hoa.process~ informations.

• Changes :

 - hoa.connect : doesn’t require arguments anymore

• Name modified objects :

 - All 2d objects are now prepended by a 2d symbol in their name, ex : hoa.wider~ => hoa.2d.wider~.
 - All 3d objects are prepended by a 3d symbol in their name, ex : hoa.3d.wider~.

 - By convention and for clarity, patchers destinated to be loaded by a hoa.process~ object are now either prepended by syn or fx. ex hoa.syn.delay~ / hoa.fx.delay~
  |=> you will need to adapt you patch in function if you used it.

• Deprecated objects :

 - hoa.convolve~ is now deprecated

 - hoa.freeverb is now deprecated

 - hoa.space~ is now deprecated. To replace it, please use the hoa.fx.gain~ abstraction loaded in a hoa.process~ object.
 

• Bugs fixed :

------------------------
version 1.0
------------------------

Functions :

- encoder : encodes a signal in the circular harmonics domain depending on an order of decomposition and an angle.
- decoder : decodes an ambisonics sound field for a circular array of loudspeakers. 
- optimBasic, optimMaxRe, optimInPhase : weights the circular harmonics signals depending to the ambisonics optimization. It can be "basic" for no optimization, "maxRe" or "inPhase".
- wider : can be used to wide the diffusion of a localised sound. The order depending signals are weighted and appear in a logarithmic way to have linear changes.
